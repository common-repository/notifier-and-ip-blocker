== Description ==

Notifier and IP Blocker is a simple plugin that can secure your web-site from spam bots and notify users. The many users, who write a comment want to know, whether the administrator had got it or not. Send a message where it is written that the comment had been received and the answer would be sent as soon as possible also if it is spam user can go to link and blocked IP.

= Features list =
* Notify a user about when he sent comment.
* Automatically block IP, if the users is informed that it is spam.
* Set span amount message.
* Manually block and unblock access by IP.
* Manually configure user letter template by editor.
* User can unlock your IP.
* Manually block IP user can't unlock.
* Manually customized Block Page content by editor.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `notifier-and-ip-blocker.zip` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Customize the message WordPress dashboard > N&IP Blocker.

== Frequently Asked Questions ==

= How can I change the message which is sent? =
You have to go WordPress dashboard > N&IP Blocker and change message content.
= Can I use my existing WordPress theme? =
Of course! Notifier and IP Blocker works out-of-the-box with nearly every WordPress theme.
= Will this work on WordPress multisite? =
Yes!
